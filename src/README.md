# Source Structure

This root-level `src` directory is reserved for project architecture and shared source documentation.

## Recommended Layout

- `src/backend` - TypeScript backend entrypoints and adapters
- `src/frontend` - TypeScript frontend entrypoints and shared UI models
- `src/admin` - TypeScript admin panel components

## Notes

Current implementation remains in `backend`, `frontend`, and `admin` folders. The new `src` directory is used for supporting architecture and future TypeScript migration planning.
